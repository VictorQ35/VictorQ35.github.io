VIRTUALIZACION

Es una tecnología que permite crear una versión virtual de un recurso o dispositivo, como un servidor, almacenamiento, red u otro recurso, dividido o particionado a través de software. Esto permite que un solo recurso físico pueda funcionar como múltiples entornos virtuales.

VIRTUALIZACION DE SERVIDOR

Consiste en dividir un servidor físico en múltiples entornos virtuales independientes, permitiendo que diferentes sistemas operativos y aplicaciones se ejecuten en el mismo hardware físico.

VIRTUALIZACION DE RED

Se utiliza para crear redes virtuales completas con topologías, segmentos y políticas de seguridad independientes, separadas de la red física subyacente.

VIRTUALIZACION DE ALMACENAMIENTO

Permite crear y administrar múltiples recursos de almacenamiento lógico a partir de uno físico, lo que facilita la gestión y el uso eficiente del almacenamiento.
